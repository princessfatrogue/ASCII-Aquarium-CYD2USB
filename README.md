# ASCII-Aquarium
ASCII Aquarium CYD is firmware for the ESP32-2432S028R Cheap Yellow Display that turns the it into a tiny animated ASCII fish tank. It renders a live aquarium scene with animated fish, bubbles, swaying seaweed, tap-to-feed food flakes, occasional octopus &amp; seahorse visitors, selectable backgrounds, preferences, optional Wi-Fi clock sync, &amp; More!
